#+title: Treemacs Cheatsheet

- n    :: /[n]/ext node
- p    :: /p/rev node
- u    :: up node
- M-n  :: next /s/ibling
- M-p  :: prev /S/ibling
- P    :: preview/peek
- RET  :: open
- or   :: open recently used window
- oaa  :: open with Ace
- th   :: toggle /H/idden files
- ov   :: open in vertical split
- oh   :: open in horizontal split
- cf   :: /n/ew file
- cd   :: /n/ew dir
- R    :: /r/ename dir/file
- H    :: collapse
- d    :: /d/elete node
- yy   :: copy path
