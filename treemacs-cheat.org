#+title: Treemacs Cheatsheet

- n    :: next node
- p    :: prev node
- u    :: up node
- M-n  :: next sibling
- M-p  :: prev Sibling
- P    :: preview/peek
- RET  :: open
- or   :: open recently used window
- oaa  :: open with Ace
- C-2 TAB :: open/expand 2 levels deep
- th   :: toggle Hidden files
- ov   :: open in vertical split
- oh   :: open in horizontal split
- cf   :: new file
- cd   :: new dir
- R    :: rename dir/file
- H    :: collapse
- m    :: mode node
- d    :: delete node
- yy   :: copy path
